Olá, para testar o código nós temos dois arquivos de texto na pasta Arquivos. Um deles é o Itens, no qual adicionamos os
itens na lista, no seguinte formato: nome, preço, quantidade. Os dados são separados por vírgula e cada novo item é uma 
nova linha. O outro arquivo é o Emails no qual adicionamos os emails, onde cada linha representa um novo email. Ambos os
arquivos já possuem uma exemplificação de como os dados devem ser descritos. Após ajustar os dados de imput nos arquivos,
basta apenas executar o arquivo Principal. Vale ressaltar que o resultado está em centavos e que para testar uma
lista vazia basta salvar os arquivos desejados sem conteúdo.
